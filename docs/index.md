# TODO LIST

Projeto desenvolvido utilizando SPEC-DRIVEN Development.

## Objetivo

O sistema permite:

- Cadastro de tarefas
- Remoção de tarefas
- Listagem de tarefas
- Definição de lembretes

## Arquitetura

O projeto utiliza:

- MVC
- Monorepo
- React
- Node.js
- Express
- MkDocs
